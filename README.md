### 参数介绍 ###

1. **countFromProps** 
	倒计时的总秒数，例如60 代表 60秒。该参数默认值为60

2. **buttonName**
	获取验证码按钮上的内容，例如请等待60s，该数字会根据countFromProps参数动态改变


### Example使用说明 ###
本地环境 **npm run start** 访问 [http://localhost:8080/index.html](http://localhost:8080/index.html "http://localhost:8080/index.html")
